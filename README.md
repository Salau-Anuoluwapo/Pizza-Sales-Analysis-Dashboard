# Pizza Sales Dashboard Analysis (Interactive Dashboard analysis using SQL & PowerBi)
## Project Objective
The store wants to analyze the pizza sales from Jan/15 – Dec/15 to gain insights into the performance of the business.

## Dataset used
- <a href="https://github.com/Salau-Anuoluwapo/Pizza-Sales-Analysis-Dashboard/blob/main/pizza_sales.csv">Dataset</a>

## Questions (KPIs)
-What is the Total revenue?
- What is the average order value?
- What is the total quantity of pizza sold?
- How many orders were placed?
- What is the average number of pizzas sold per order?
- What are the daily trends for total orders?
- What is the monthly trend for total orders?
- Percentage of sales by pizza category
- percentage of sales by pizza size
- Total pizza sold by each category
- Top 5 pizza by revenue, quantity & order
- Bottom 5 pizza by revenue, quantity & order

- Dashboard interaction
